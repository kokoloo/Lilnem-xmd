## 🏗️ Installation Local

### 🔧 Prérequis
- Node.js v18+
- Git
- FFmpeg (optionnel)

### 📝 Instructions

```bash
# 1. Cloner le repository
git clone https://github.com/kokoloo/Lilnem-xmd
cd Lilnem xmd

# 2. Install dependencies  
npm install

3. Configure variables (.env)  
cp .env.example .env  
Edit .env with your information

4. Start the bot  
npm start
```

🔧 Troubleshooting

❌ Bot not connecting
- ✅ Check your *SESSION_ID*  
- ✅ Regenerate if necessary  
- ✅ Check environment variables

🔄 Bot disconnects
- ✅ Restart the service  
- ✅ Check error logs  
- ✅ Update your *SESSION_ID*

📱 Commands not working
- ✅ Check the *PREFIX* (default: `.`)  
- ✅ Check the bot *MODE*  
- ✅ Make sure you're in an authorized group
